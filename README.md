# Work Automation AI Tools

このリポジトリは、生成AI（ChatGPT）を活用して作った  
**業務効率化・業務自動化ツール（Python、VBA）** をまとめたものです。

- 社内の事務作業  
- Excelの集計・整形  
- メールの読み取り・返信  
- 承認フロー管理  
- フォルダ整理  
- CSV加工  
など、毎日の作業を自動化するためのツールを公開します。

## 目的
- 自分のスキルを体系的にまとめる
- 副業・ポートフォリオとして活用する
- 将来のフリーランス活動に向けて資産を残す

## 構成
- docs/ → 思想・設計・計画・備忘録
- projects/ → 完成した自動化ツールの本体（主役）

## 使用技術
- Python
- Excel VBA
- ChatGPT（コード生成・設計補助）
