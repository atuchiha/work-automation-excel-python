AI × VBA × Python を活用した業務自動化ツールの開発計画

このドキュメントでは、今後どのように自動化ツールを増やし、
どんなスキルを伸ばしていくかをまとめています。

目標や計画を見える化することで、
迷わずに進めることを目的としています。

🎯 1. 全体目標（General Goals）

AI・VBA・Python を使った「実務で使える自動化ツール」を増やす

同じ作業をくり返さない仕組みを作る

シンプルで誰でも使えるツールを作る

コードの質を上げ、保守しやすくする

副業やポートフォリオに活用できる成果物を作る

📌 2. 開発ステップ（Development Phases）
Phase 1：基盤づくり（準備）

リポジトリ構成を整える

overview と roadmap を作成

projects フォルダのテンプレートを用意

AI を使った開発ルールを決める

Phase 2：小さなツールから作り始める

小規模で効果の高い自動化を開発

VBA や Python の基本処理を組み立てる

シンプルなメール通知ツール

小さなデータ整理ツール

Phase 3：実務レベルの自動化ツール開発

ファイル整理の完全自動化

Excel・CSV の処理パイプライン自動化

データ変換・清掃の一括処理ツール

承認フローやメール通知の自動化

Web 情報取得の自動化など

Phase 4：AI と連動した高度な自動化

AI による判断・説明文生成

自然言語でツールを動かせる仕組み

自動化フローのAI解析・改善

AI × Python のハイブリッドツール

Phase 5：改善と最適化

コードのリファクタリング

エラー処理やログ機能の追加

実行速度の改善

ファイル構成の見直し

ユーザー向け説明の改善

📅 3. 開発予定リスト（Backlog）

ここは自由に書き足せます。
例として、よくあるニーズを入れています。

Excel・CSV の自動整形ツール

ファイル名の自動リネーム

フォルダ仕分けロボット

メール自動通知（Python または VBA）

Web 情報取得と自動保存

PDF 変換 & 統合ツール

日報データの自動集計

生成AIによる説明文自動生成

必要に応じて消したり追加してください。

📈 4. 学習・スキルアップ計画

ChatGPT を使った AI 開発スキル向上

VBA の基本～中級の復習

Python のデータ処理・ファイル操作を強化

GitHub の運用スキルアップ

自動化ツールの保守・改善スキル

🚀 5. 目指す姿（Vision）

毎日の作業が自動化され、時間が大きく節約される

プログラミング初心者でも使えるツールが揃う

実務にも副業にも使える「自動化ツールの資産」を作る

AI と人が協力して働くスタイルを実現する
